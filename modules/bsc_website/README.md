# BSC International Website Module

Odoo 18 website module that replicates the Groupe BSC International corporate website.

## Features

- **Modern Responsive Design** - Mobile-first approach with Bootstrap 5
- **Homepage** - Hero slider, services preview, team preview, CTA sections
- **Services Page** - Detailed service cards with features
- **About Page** - Company info, mission, vision, values
- **Team Page** - Team members organized by office location
- **Contact Page** - Contact form integrated with CRM, Google Maps
- **Custom Header/Footer** - Branded navigation and footer

## Services Included

1. Business Connection
2. Investment Facilitation
3. Consultancy & Advisory
4. Business Aviation
5. Business Events
6. Company Formation

## Installation

1. Copy this module to your Odoo addons directory
2. Update the apps list
3. Install "BSC International Website"
4. Upload images to `/bsc_website/static/src/img/`

## Required Images

Place the following images in `static/src/img/`:

- `logo.png` - Company logo
- `about-img.jpg` - About section image
- `about-company.jpg` - About page image
- `pattern.png` - Background pattern (optional)
